# key_value_searcher
Handler of requests for values, accessible by keys and located in an external file.
## Request a value by key  
Query example  
```
curl http://localhost/kvs/app/ -d 'key=TL&filename=../documents/test.txt'
```  
Result  
```json
{
   "key" : "TL",
   "value" : "LT"
}
```
